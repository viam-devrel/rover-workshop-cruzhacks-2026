# CruzHacks 2026 Viam Rover Workshop
Welcome CruzHacks 2026 Viam Rover workshop attendees! This repo contains the steps you'll follow alongside live instruction from @AdrienneTacke. You can always reference this repo in case you need a little more time or need to return to a particular step. Please let us know if you encounter any mistakes!

This workshop will walk you through how to set up a Viam Rover and start controlling it through the Viam platform.

## Prerequisites
- A Viam account, and then [sign in](https://app.viam.com/)
- [Raspberry Pi Imager](https://www.raspberrypi.com/software/) installed
- A microSD card and adapter, if needed
- 4 (four) charged 18650 batteries (provided at workshop)
- A Viam Rover (provided at workshop)
